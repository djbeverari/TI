Manual de Uso — Painel Data Sync
Guia para operadores e supervisores

Endereço do painel: http://192.168.0.147:8080/painel.html
Abra este endereço em qualquer navegador de qualquer computador da rede.

O painel atualiza automaticamente a cada 30 segundos. Não é necessário recarregar a página manualmente.

1. Como Acessar o Painel

1
Abra o navegador (Chrome, Edge, Firefox)
2
Digite na barra de endereço: http://192.168.0.147:8080/painel.html
3
O painel carrega imediatamente — sem login necessário
Se o painel não abrir: o servidor pode estar desligado ou o serviço parou. Entre em contato com a TI.

4. Horários de Sincronização

HorárioDiasObservação
10:30Segunda a sextaCiclo principal — lojas abertas, melhor taxa de sucesso
14:30Segunda a sextaCiclo intermediário
16:30Segunda a sextaÚltimo ciclo — lojas de shopping podem estar encerrando
Sábados e domingos: a automação não executa. Os agendamentos ficam ativos mas o script bloqueia a execução nesses dias.

Por que algumas lojas falham no ciclo das 07h/08h (testes ou manutenção)?

Lojas que ainda não abriram têm o computador desligado ou sem conexão. Isso causa timeout (4 minutos de espera) e a loja aparece como falha. É comportamento normal — o ciclo das 10:30 tentará novamente quando as lojas estiverem abertas.

5. O que Fazer Quando Há Falhas

Poucas lojas falharam (1 a 5)

Situação normal, especialmente no ciclo das 16:30 quando algumas lojas de shopping encerram mais cedo. O próximo ciclo tentará novamente. Só investigar se a mesma loja falhar em 3 ciclos seguidos.

Muitas lojas falharam (mais de 10)

Possíveis causas: queda de internet, servidor sobrecarregado, horário fora do expediente das lojas (ciclo das 16:30 com lojas fechando). Verificar se o problema persiste no ciclo seguinte.

Todas as lojas falharam

Problema grave. Possíveis causas: servidor sem internet, erro no script, serviço do Linx DataSync parado. Acionar a TI imediatamente.

Uma loja específica falha sempre

Se a mesma loja falha repetidamente mesmo nos ciclos das 10:30 e 14:30, as causas mais comuns são:

  • Computador da loja desligado ou sem internet
  • Atalho do DataSync inválido ou desconfigurado para essa loja
  • Problema específico no DataSync client da loja

Neste caso, acionar suporte para verificar a loja presencialmente ou por acesso remoto.

6. Controles Manuais (para o operador no servidor)

Na área de trabalho do servidor (192.168.0.147, usuário Datasync), existem três atalhos:

ArquivoQuando usarComo usar
pausar-agendamentos.bat Quando precisar fazer manutenção no servidor e não quer que o sync rode automaticamente Clique duplo — confirma com "S" se solicitado
retomar-agendamentos.bat Após a manutenção, para reativar os ciclos automáticos Clique duplo
executar-datasync.bat Quando precisar rodar uma sincronização fora do horário agendado Clique duplo — o sync inicia em background, acompanhe pelo painel
Atenção: após usar pausar-agendamentos.bat, lembre-se sempre de usar retomar-agendamentos.bat para reativar os ciclos automáticos.
