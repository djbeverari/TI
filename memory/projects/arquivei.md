# Project: Arquivei

**Project name:** Arquivei  
**Also called:** Document archival system  
**Status:** Active development  
**Last update:** 2025-05-14 (network accessibility version)

## What It Is
A document/invoice processing and archival system for Brazilian NFe (electronic invoices) and related documents. Parses XML invoices, extracts data, and provides a web-based UI for data management.

## Tech Stack
- **Backend:** Python (data processing, extraction, organization)
- **Frontend:** HTML/CSS (web interface)
- **Data format:** XML (NFe invoices)
- **Version control:** Git

## Current Focus Areas (from recent commits)
1. **Field positioning** — aligning CPF, CEP, and other form fields correctly
2. **Monthly organization** — separating data by month
3. **UI/Color updates** — HTML/CSS refinements with correct colors
4. **Network accessibility** — ensuring access from any machine on network

## Key Brazilian Document Fields
- **CPF** — Brazilian individual tax ID
- **CEP** — Postal code
- **NFe** — Electronic invoice

## File Structure
- `Desktop/Arquivei/` — Main project directory
- `Desktop/Arquivei/templates/` — HTML/CSS templates
- `Desktop/Arquivei/Arquivei Citados/` — Archive of processed documents

## Recent Changes
- Fixed field spacing and positioning
- Updated HTML/CSS with correct colors
- Implemented monthly separation
- Improved network accessibility
