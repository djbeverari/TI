# IT Support Context

**Role:** Daniella handles IT troubleshooting and support for Dorinhos organization.

## Common Issues & Solutions

### Microsoft Office 365 / Outlook
- **PDF Opening in OWA** — Antivirus (Kaspersky) may block inline PDF viewing
  - Solution: Add `outlook.office.com` to antivirus whitelist
  - Symptoms: "não foi possível concluir a solicitação" error
  - Affects: Multiple accounts with same antivirus policy

### Antivirus Configuration
- **Primary:** Kaspersky Security Center
- **Location:** Whitelist in Web Control settings
- **Process:** Kaspersky Admin Console → Policies → Web Control → Exceptions

## Support Workflow

1. **Diagnose** — Identify symptoms and affected systems
2. **Research** — Check if it's antivirus, policy, or infrastructure
3. **Test** — Reproduce on different accounts/devices
4. **Document** — Record solution in KB article
5. **Communicate** — Draft response with step-by-step fix
