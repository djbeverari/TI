# TI (Tecnologia da Informação)

Infraestrutura, suporte e automação de TI da Dorinhos.

## Estrutura

- `CLAUDE.md` — Configuração de skills e preferências
- `MEMORY.md` — Índice de memória
- `memory/glossary.md` — Termos de TI, processos e suporte

## Como Usar

Abra este projeto no Claude Code. Ele carregará automaticamente `CLAUDE.md` com as skills específicas para gestão de TI.

## Conteúdo

- Código de automação (Python, PowerShell)
- Documentação de infraestrutura
- Procedimentos de suporte
- Sistema Arquivei (processamento de documentos)

## Padrões

- Automação e scripting
- Suporte orientado por tickets
- Análise de métricas de TI (uptime, performance, segurança)
